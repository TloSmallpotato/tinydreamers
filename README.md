
# Kids Learning Tracker

<!-- This mobile app helps parents track their children's reading progress, vocabulary development, and learning moments -->

A React Native + Expo mobile application for tracking children's books, words, and learning moments.

## Features

- Track books read by children
- Monitor vocabulary and words learned
- Record video moments of learning
- Profile management for multiple children
- Subscription-based premium features

## Tech Stack

- React Native
- Expo 54
- Supabase (Backend)
- TypeScript
